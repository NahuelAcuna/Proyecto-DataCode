HTML

HyperText Markup Language


HTML (HyperText Markup Language) es, según sus siglas en inglés, un lenguage de marcado de hipertexto. ¿Qué es esto? Bueno, basicamente nos ayuda a generar nuestra estructura básica de nuestro sitio web y el contenido que será mostrado al público.

- ¿Qué es exactamente lo que definimos? En el HTML generamos las etiquetas que indicarán el orden en el que estará distribuido nuestro sitio web, dentro de ellas, definimeros nuestro texto legible para posteriormente ser modificado y estilado con CSS utilizando dichas etiquetas.

- ¿Esto quiere decir que HTML no da estilos al sitio? Exactamente. Una vez generado el contenido y la estructura del sitio, pasaremos a brindar estilos, apariencia, etc con CSS.


Bien, arranquemos con un poco de HTML:

Ya teniendo en cuenta que definiremos etiquetas y que deben mantener un orden para respetar y conseguir esa estructura básica, pasemos a ver como se realiza.

La estructura básica es como una noticia la cual contiene un titulo, cuerpo y pie de página. Nosotros a esto lo definimos como Header, container y footer como ya podran haber visto en casi todos los sitios webs que existen.

<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta http-equiv="X-UA-Compatible" content="IE=edge">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Hola Mundo!<title>
</head>
<body>


</body>
</html>

Teniendo en cuenta la estructura brindada, paso a explicar: La primera etiqueta inicial define que trabajaremos sobre un documento HTML.

Luego comienza la etiqueta HEAD, todo lo que se contiene dentro de esta etiqueta no es visible para el usuario y se utiliza para definir parámetros con los cuales se construye el sitio web, ej: Linkear nuestro CSS, JS, titulos, palabras claves, idiomas, etc.

Ahora comienza BODY, dentro de esta etiqueta es con la que más trabajaremos ya que es en la que definimos todo nuestro contenido visible para el cliente y la que luego, con cada etiqueta, aplicaremos los estilos.

- Ahora, seguro te preguntarás, ¿donde está el "pie" del sitio? Bueno, dentro de nuestro body se utilizarán etiquetas semánticas que ayudaran a definir la estructura que anteriormente mencioné. Estas son: header, main y footer.

Con esta base ya podes comenzar a trabajar con HTML, pero antes, te recomiendo ver como es la sintaxis de HTML y poder continuar con este gran curso!
